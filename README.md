# 💬 Real-Time Chat Application (Flask + SocketIO)

A secure **real-time chat application backend** built using **Flask**, **JWT Authentication**, and **Socket.IO**.  
This project supports user authentication, private messaging, and live message delivery.

---

## 🚀 Features

✅ User Signup & Login  
🔐 JWT Authentication  
💬 Real-time private messaging (Socket.IO)  
👥 Fetch all users (except logged-in user)  
📜 Chat history between two users  
🗄 SQLite database integration  
🔒 Password hashing using Werkzeug  
🌍 CORS enabled for frontend integration  

---

## 🛠 Tech Stack

- **Backend:** Flask
- **Authentication:** Flask-JWT-Extended
- **Real-Time Communication:** Flask-SocketIO
- **Database:** SQLite
- **Security:** Werkzeug Password Hashing
- **Environment Management:** Python Dotenv
- **CORS Support:** Flask-CORS
